👥 Grupos com este bot
📱 Como adicionar ao grupo
  1. Abra o BoresChat no celular
  2. Entre no grupo como administrador
  3. Toque em Gerenciar Grupo → Bots
  4. Cole seu token:
  5. Toque em Adicionar Bot
  6. Pronto! Digite /ajuda no grupo para testar 🔥
🤖 Comandos padrão do bot
/ajudaLista todos os comandos disponíveis
/menuMenu interativo com botões clicáveis
/pingTesta se o bot está online
